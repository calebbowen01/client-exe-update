# Assets Directory

This directory contains all the resource files used by the Anomaly.exe application.

## Directory Structure

- **images/** - Image files (PNG, JPG, etc.)
- **audio/** - Sound effects and music files
- **config/** - Configuration files
- **data/** - Data files and other resources

## Usage

Place your resource files in the appropriate subdirectories. The Anomaly.exe application will look for assets in this folder structure.

## Adding New Assets

1. Identify the type of asset you want to add
2. Place it in the corresponding subdirectory
3. Ensure the file is properly named and formatted
4. Update any configuration files if necessary
