# Client EXE Update

This repository contains the Anomaly.exe application and its associated resources.

## Repository Structure

- **Anomaly.exe** - The main executable application
- **assets/** - Directory containing all resource files
  - **images/** - Image assets
  - **audio/** - Audio files
  - **config/** - Configuration files
  - **data/** - Data files

## Getting Started

1. Download or clone this repository
2. Place any additional assets in the appropriate `assets/` subdirectories
3. Run `Anomaly.exe` to start the application

## Assets

All resource files should be placed in the `assets/` directory. See [assets/README.md](assets/README.md) for more information about the directory structure and usage.
